How it maps to the real app: the mesh is a single fixed layer on Layout.vue's root bg-canvas element (z-index 0), exactly like the login screen's .login__mesh. List rows and the content area are already transparent, so the gradient shows through the listing's negative space. Grid/gallery cards and the sidebar are opaque by default (gradient peeks between cards) — toggle Sidebar → Translucent to preview letting it bleed through the sidebar too. Intensity previews the layer opacity knob (I'd ship a per-user toggle defaulting to Subtle). No blur() filter is used — soft radial edges only — to keep paint cheap on scroll.
