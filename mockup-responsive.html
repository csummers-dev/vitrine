≥ 1280 · sidebar 256 · search 380 · InfoPane inline · 5-col list

< 540 · all above + list 3-col (–size, –modified, meta merged into name cell) · InfoPane → full-width sheet
Highlighted with amber outlines in each mockup are the elements that changed at that breakpoint.
